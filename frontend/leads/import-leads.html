Import Leads from CSV

Upload a CSV file containing your leads. Ensure the file follows the required format for successful import.

CSV Format

Your CSV file should include the following columns in the specified order:

full_name

Full Name

email

Email Address

company_name

Company Name

job_title

Job Title

phone_number (Optional)

Phone Number (e.g., +1-555-123-4567)

linkedin_url (Optional)

LinkedIn Profile URL
